# PDF Editor

ブラウザ上で動作するシンプルなPDFエディタです。ページの複製、削除、複数のPDFの結合などの基本的な編集機能を提供します。

## 機能

- ページの複製
- ページの削除
- 複数のPDFの結合
- 編集したPDFのダウンロード

## 技術スタック

- HTML5
- CSS3
- JavaScript
- PDF.js (PDFの表示用)
- PDF-Lib (PDFの編集用)

## セットアップ

1. リポジトリをクローン:
```bash
git clone https://github.com/motoya0118/edit_pdf_made_in_cursor.git
cd edit_pdf_made_in_cursor
```

2. 必要なパッケージをインストール:
```bash
npm install
```

3. ローカルサーバーを起動:
```bash
python3 -m http.server 8000
```

4. ブラウザで `http://localhost:8000` にアクセス

## 使い方

1. PDFファイルをアップロード
2. 編集したいページを選択（複数選択可）
3. 必要な操作を実行（複製、削除、結合）
4. 編集が完了したら「ダウンロード」ボタンで保存

## 特徴

- ブラウザ上で完結する処理
- 直感的なユーザーインターフェース
- リアルタイムプレビュー
- 無料で利用可能

## 注意事項

- 大きなPDFファイルの処理には時間がかかる場合があります
- ブラウザの互換性に注意が必要です
- 処理中の操作は控えめにしてください
